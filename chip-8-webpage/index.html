Chip-8 Emulator

This is a Chip-8 emulator created entirely in JavaScript (TypeScript) by 5 students at Simon Fraser University for their project for CMPT 276 - Introduction to Software Engineering.

This project consists of 4 main components:

  • Emulator
  • Visualizer
  • Tool
  • Games

This is a semester long project and we will have 4 different releases. We aim to have the project completed by the third release.
